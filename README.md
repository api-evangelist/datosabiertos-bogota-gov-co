# Datos Abiertos Bogota (datosabiertos-bogota-gov-co)

Datos Abiertos Bogota is a municipal government open-data portal for Colombia running **CKAN**, exposing a standardized catalog API over ~2,112 datasets.

**URL:** [Visit APIs.json URL](https://raw.githubusercontent.com/api-evangelist/datosabiertos-bogota-gov-co/refs/heads/main/apis.yml)

**Run:** [Capabilities Using Naftiko](https://github.com/naftiko/fleet?utm_source=api-evangelist&utm_medium=readme&utm_campaign=datosabiertos-bogota-gov-co-api-evangelist&utm_content=repo)
